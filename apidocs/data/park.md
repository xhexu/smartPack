#### 园区实体  

````
id : 主键
code : 园区编号
parkName : 园区名称
parkAddr : 园区地址
prpEnterpriseName : 产权方企业名
prpRegTime : 产权方注册时间
prpLinkmanName : 产权方联系人
prpLinkmanPhone : 产权方联系电话
parkPrincipalName : 园区负责人
parkPrincipalPhone : 园区负责人联系电话
parkEmail : 园区电子邮箱
serviceTime : 投入使用时间
propertyType : 物业类型
floorArea : 占地面积,平方米
coveredArea : 建筑面积,平方米
floorA : 办公楼A,楼层
floorB : 办公楼B,楼层
enterpriseIntro : 企业简介
remark : 备注
active : 是否有效 : Y/N
createTime : 创建时间
createUser : 创建人
updateTime : 修改时间
updateUser : 修改人
````
